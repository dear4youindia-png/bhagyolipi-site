# Bhagyolipi Website

এই static website টি `bhagyolipi.com`-এ GitHub Pages দিয়ে live করার জন্য তৈরি।

## Files

- `index.html`
- `styles.css`
- `script.js`

## GitHub Pages Setup

1. GitHub-এ একটি new repository তৈরি করুন।
2. এই project-এর সব file সেই repo-তে upload বা push করুন।
3. Repository settings থেকে `Pages` enable করুন।
4. Source হিসেবে main branch root select করুন।
5. Custom domain হিসেবে `bhagyolipi.com` দিন।

## Dear4You APK Setup

1. GitHub repo-র `Releases` section-এ যান।
2. একটি নতুন release publish করুন।
3. `Dear4You.apk` file-টি release asset হিসেবে upload করুন।
4. Release download link এই format-এ ব্যবহার করতে পারবেন:

   `https://github.com/your-username/your-repo/releases/latest/download/Dear4You.apk`

5. `script.js`-এ থাকা placeholder GitHub username/repo name বদলে দিন।

## Change Text Later

অ্যাপের নাম, বর্ণনা, button text, support section ইত্যাদি `index.html` থেকে সহজে edit করা যাবে।
